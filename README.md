# Data Analysis Project

Data Analysis & Management final project workspace focused on answering a sports analytics question with SQL, Python, and visualizations.

## Project overview

Question: Which NBA team has the highest percentage?

This project will use a publicly available NBA dataset, load it into a SQL database (MySQL or SQLite), clean the data, perform analysis in Python (Pandas/NumPy), and create visualizations (Matplotlib/Seaborn) to support the final report and presentation.

## Grading Rubric

Grading is based on completion of each category.

| Category              | What We're Looking For                                                                                                |
| --------------------- | --------------------------------------------------------------------------------------------------------------------- |
| Data Collection & SQL | Dataset imported into a SQL database (MySQL or SQLite); `SELECT`, `WHERE`, `ORDER BY` used; `JOIN` if multiple tables |
| Data Cleaning         | Evidence of checking for and handling duplicates, NULLs, bad values, and unnecessary data                             |
| Data Analysis         | At least 4 meaningful analyses using Pandas/NumPy; clear connection between analysis and question                     |
| Data Visualization    | At least 3 visualizations with 3+ chart types; proper labels/titles; Seaborn usage                                    |
| Data Report           | Complete report with intro, dataset overview, cleaning summary, findings, and conclusion                              |
| Presentation          | Clear delivery; shared question, data, key findings, and conclusion; answered questions                               |

## Project Checklist

Use this checklist to make sure you have everything before you present.

### Getting Started

- [ ] I have chosen a topic/question that I find interesting
- [ ] I have found at least one dataset that relates to my question
- [ ] I have imported my dataset into a SQL database (MySQL or SQLite)

### Data Cleaning

- [ ] I checked for duplicate rows and handled them
- [ ] I checked for NULL or missing values and handled them
- [ ] I fixed any problematic or inconsistent values
- [ ] I removed unnecessary columns or rows

### Analysis

- [ ] I connected to my database from Python (`pymysql`/`sqlalchemy` for MySQL, or `sqlite3` for SQLite)
- [ ] I pulled my data into a Pandas DataFrame
- [ ] I performed at least 4 distinct analyses
- [ ] I can explain what each analysis tells me

### Visualization

- [ ] I created at least 4 visualizations
- [ ] I used at least 3 different chart types
- [ ] All charts have titles and labeled axes
- [ ] At least one chart uses Seaborn

### Report & Presentation

- [ ] My data report includes: Introduction, Dataset Overview, Cleaning Summary, Analysis & Findings, and Conclusion
- [ ] My visualizations are embedded in the report with explanations
- [ ] I am ready to present for 3-5 minutes and answer questions
